About the JURAL Project

The JURAL project was created as part of an experiment exploring the practical limits of so-called “AI technologies” in real-world software development.

This repository hosts a growing collection of free console utilities for Windows, developed with the assistance of ChatGPT. The primary goals are:

  • improving performance and functionality of standard tools;
  • exploring low-level aspects of working with files and devices;
  • optimizing speed and minimizing system overhead.

Author of the project — Iurii Mykhoparkin.

Implemented utilities

  • Search for text fragments in large text files (encoding-independent).
  • Search for binary fragments in large binary files.

Planned utilities

  • Positional and context-based insert/replace operations for text and binary files.
  • Advanced filtering of text and binary files.
  • Duplicate line detection/removal in text files.
  • Text file comparison based on custom substring rules.
  • Transliteration tools.
  • Frequency dictionary generation.
  • Advanced binary file dump analyzer.
  • CSV → table conversion.
  • CSV comparison by arbitrary column sets.
  • Detection of duplicates in CSV by multiple fields.
  • CSV filtering by arbitrary combinations of fields.

Repository structure

Find_Bin_Frag/   – binary fragment search utility  
Find_Txt_Frag/   – text fragment search utility
